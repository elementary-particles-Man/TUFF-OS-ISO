## TUFF-OS について
The Ultimate Fortress Foundation OS (TUFF-OS) は、物理層での絶対的なデータ主権を提供するセキュリティ特化型 OS です。

## 配布
- **バージョン**: 1.1.0 (インストーラ起動版)
- **ターゲット**: x86_64 UEFI Secure Boot 準拠
- **最新 ISO**: `latest/TUFF-OS-latest.iso`
- **ISO生成**: 標準ツールである `tools/tuff-builder` でリリース構成の ISO を生成し、手作業での組み立てや編集は行いません。
